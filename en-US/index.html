Abstract
Welcome to the home site of the UKRISS Crosswalk Connector. This document describes how to install, implement and configure the tool for the Crosswalk project.
The Crosswalk Connector is a demonstration tool to show how support can be given for extracting research data from a university's databases and exporting it to XML conformant with the CERIF standard. The tool is a general purpose data integration tool with specialised steps for handling CERIF.
The Crosswalk Connector is available as a download from this site. It is open-source, and is freely available to the public under GPL3.
